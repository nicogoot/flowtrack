Guía de configuración
1

Conseguir Groq API Key

Entrá a console.groq.com → Sign Up con tu cuenta Google → "API Keys" → "Create API Key". Copiala y pegala arriba. Completamente gratis.

2

Crear tu Google Sheet

Abrí sheets.google.com y creá una planilla nueva. En la fila 1 poné estos encabezados exactos:

Fecha TRX | Monto | Moneda | En (Descripción) | Comentario | Con (Medio de Pago) | Cuota | Cuotas | Cantidad | Precio

3

Crear el Apps Script (el puente)

En tu Sheet: menú Extensiones → Apps Script. Borrá el código default y pegá el código del archivo script.gs que te dimos. Guardá → Ejecutar → Implementar → Nueva implementación → Aplicación web → "Cualquier persona" → Implementar. Copiá la URL y pegala en el campo de arriba.

4

Deploy en Netlify (para el celu)

Entrá a netlify.com/drop, arrastrá el archivo flowtrack.html. En segundos tenés una URL pública. Abrila desde el iPhone → compartir → "Agregar a pantalla de inicio".

5

Conectar Power BI

En Power BI Desktop: Obtener datos → Google Sheets → pegás la URL de tu planilla. Listo. Se actualiza cada vez que guardás desde la app.
